## Installation:

### Requirements:

- [x] python 3.9 or later
- [x] PyQt5 5.15.10 or later

```
pip install git+https://github.com/Deadbush225/RePhrase
```

### Updates:

```
pip install --upgrade git+https://github.com/Deadbush225/RePhrase
```

## Usage:

Go to any terminal and run:

```
python -m rephraser
```
